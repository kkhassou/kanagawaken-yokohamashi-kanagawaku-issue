# 神奈川区 地域の困りごと報告プラットフォーム

神奈川県横浜市神奈川区の地域の困りごとを収集・管理するためのプラットフォームです。

## 概要

地域住民が日常で感じる困りごと（危険箇所・街灯不足・道路の問題など）を報告し、対応状況をカンバン形式で管理します。

## 困りごとの例

- 手すりがなくて危ない
- カーブミラーがなくて危ない
- 夜、街灯が少なくて暗くて怖い
- 歩道が狭い・段差がある
- ゴミの不法投棄がある

## 使い方

### 困りごとを報告する

**Google フォームから投稿できます（GitHub アカウント不要）：**

> [困りごと報告フォーム](https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform)
>
> ※ フォームのリンクは準備完了後に更新されます

フォームから投稿された内容は、自動的に GitHub Issue として登録されます。

### 対応状況を確認する

**誰でもアクセスできる公開ページ：**

> [困りごと一覧ページ](https://kkhassou.github.io/kanagawaken-yokohamashi-kanagawaku-issue/)

GitHub アカウントがなくても、すべての困りごとの一覧と対応状況を確認できます。

## システム構成

```
Google Form → Google Apps Script → GitHub Issue → GitHub Project (カンバン)
                                        ↓
                                  GitHub Pages (公開ビューア)
```

| コンポーネント | 役割 |
|---------------|------|
| Google Form | 困りごとの投稿フォーム |
| Google Apps Script | フォーム → Issue 自動連携 |
| GitHub Issues | 困りごとの管理 |
| GitHub Projects | カンバン形式で対応状況管理 |
| GitHub Pages | 誰でも見れる公開ビューア |

## カテゴリ

| ラベル | 内容 |
|--------|------|
| `安全` | 手すり・ガードレール・転落防止など |
| `街灯・照明` | 街灯不足・暗い場所 |
| `道路・歩道` | 道路損傷・歩道の問題・段差 |
| `カーブミラー・標識` | ミラーの設置・標識の問題 |
| `公園・緑地` | 公園の設備・緑地の管理 |
| `ゴミ・不法投棄` | 不法投棄・ゴミ問題 |
| `その他` | 上記以外の困りごと |
